Language Learning Application

We will be developing a language learning application for early education students and their instructors to use. The application will prompt the student to use syllables in order to form words and sentences that describe a picture displayed on the screen. The instructor will be able to add new combinations of pictures and syllables for the student to work with using a intuitive creator.

Key Features

Complete words and sentences in foreign languages by matching language syllables together, in order to describe pictures.

Add new challenges for students using the intuitive creator interface. Choose your own pictures and syllables.
